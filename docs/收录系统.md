## 安装和使用

1. 通过官网下载本系统源代码或克隆本仓库
2. 将代码上传到您的Web服务器上面。
3. 在数据库中创建一个新的数据库，并将`create_table.sql`文件导入该数据库中。
4. 访问**你的域名/install.php**输入相关信息点击提交即可。
5. 在浏览器中打开网站地址，即可使用。
6. 后台地址为：域名`/admin`。
